### Git 学习

**GIT3.1 分支基础**（✅ 掌握）
- 分支本质：指向 commit 的指针（.git/refs/heads/ 里的文件）
- HEAD 指针：指向当前所在分支（.git/HEAD 里的文本）
- `git branch 分支名` — 创建分支
- `git switch 分支名` — 切换分支
- `git merge 分支名` — 合并分支
- 合并冲突的产生与解决（<<<<<<< ======= >>>>>>> 标记）
- `git branch -d 分支名` — 删除合并后的分支
- `git log --oneline --graph --all` — 图形化查看分支

**GIT4.1 远程仓库操作**（✅ 掌握）
- SSH 密钥生成与 GitHub 配置
- `git remote add origin 地址` — 添加远程仓库
- `git remote -v` — 查看远程仓库
- `git push -u origin main` — 推送并设置上游
- `git push` — 简写推送（设过 -u 后）
- `git pull origin main` — 拉取远程更新并自动合并
- `git fetch origin main` — 只下载不合并
- `pull = fetch + merge`
- `origin/main` — 远程跟踪分支（本地存的远程镜像）
- 理解 `.git/config` 中的 `[remote "origin"]` 和 `[branch "main"]` 配置
